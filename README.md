# Promptsikho Tools

A minimal React 19 project with Vite, TypeScript, Tailwind CSS, and shadcn/ui components.

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:5173`

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React 19
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui

## Add More Components

```bash
npx shadcn-ui@latest add <component-name>
```

Example: `npx shadcn-ui@latest add input`
